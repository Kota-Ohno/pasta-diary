# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 18g
- にんにく 10g
- 塩 少々
- 唐辛子 少々
- 魚醤 3g
- ホタルイカ 70g
- つまみ菜 60g

---

# 作り方

1. にんにくをスライス、つまみ菜を一口大にカットする。ホタルイカの掃除をしておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルと唐辛子を冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. つまみ菜とホタルイカを加え、温める。
5. パスタがアルデンテになったらフライパンにゆで汁と魚醤を加え、中火にかけてパスタと和える。
6. 皿に盛り付ける。
