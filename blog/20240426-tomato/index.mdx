# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 18g
- 塩 少々
- ニンニク 10g
- ベーコン 45g
- たまねぎ 70g
- トマトソース 60g
- ミニトマト 60g

---

# 作り方

1. ニンニクをみじん切り、ベーコンを短冊切り、玉ねぎをスライス、ミニトマトを半割にする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ミニトマトを断面を下にして加え、しっかりと焼き付ける。
5. ベーコンを加え、焼き目をつける。
6. たまねぎがしんなりする程度まで炒める。
7. パスタがアルデンテになったらフライパンに加え、中火にかけてパスタと和える。
8. 皿に盛り付ける。
