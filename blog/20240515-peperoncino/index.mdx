# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 18g
- にんにく 10g
- 黒ニンニク 2 かけ
- 塩 少々
- 唐辛子 少々
- アンチョビ 5g
- タコ 60g

---

# 作り方

2. にんにくをスライス、タコを一口大にカットする。
3. 1.5%の濃度の塩水でパスタを茹でる。
4. スライスしたニンニクとオリーブオイルと唐辛子を冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
5. ゆで汁と黒ニンニク、アンチョビを加え、煮溶かす。
6. パスタが上がる３０秒前にタコを加えて温める。
7. パスタがアルデンテになったらフライパンにゆで汁と魚醤を加え、中火にかけてパスタと和える。
8. 皿に盛り付ける。
