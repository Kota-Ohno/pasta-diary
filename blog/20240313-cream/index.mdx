# 材料

- スパゲティ 100g
- バター 20g
- 生クリーム 40g
- 塩 少々
- パルミジャーノレッジャーノ 15g
- ベーコン 45g
- たまねぎ 100g
- 黒胡椒 1g

---

# 作り方

1. ベーコンを短冊状に、玉ねぎをスライスでカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ベーコンに焼き目をつけ、玉ねぎがしんなりするまで炒める。
4. バターと生クリームと 20g の水を加え、バターを溶かす。
5. パスタがアルデンテになったら黒胡椒をフライパンに加え、チーズと合わせてパスタと和える。
6. 皿に盛り付ける。
