# 材料

- スパゲティ 100g
- たらこ 40g
- めんつゆ(創味のつゆ) 5g
- バター 15g
- 生クリーム 50g
- 黒胡椒 少々

---

# 作り方

1. たらこは皮が気にならなくなる程度に叩いておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ボウルにめんつゆ、バター、生クリームを加える。
4. パスタ湯の蒸気でボウルを温め、バターを溶かす。
5. ボウルにたらこと黒胡椒を加える。
6. パスタがアルデンテになったらボウルにパスタを加えて和える。
7. 皿に盛り付ける。お好みで粉チーズを振りかけたり卵黄をのせる。
