# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- エキストラバージンオリーブオイル 20g
- びんちょうマグロ 70g
- 鮎の魚醤 3g
- イタリアンパセリ 少々
- 塩 1g

---

# 作り方

1. びんちょうマグロを食べやすい大きさにカットし、塩・オリーブオイル・鮎の魚醤・イタリアンパセリと和えてマリネしておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. パスタがアルデンテになったらニンニクオイルの入ったフライパンを中火にかけ、フライパンにパスタとマリネしておいたマグロとゆで汁少々を加える。
5. マグロの表面が少し白みがかるまでパスタを混ぜながら加熱し、皿に盛り付ける。
