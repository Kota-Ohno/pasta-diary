# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- エキストラバージンオリーブオイル 20g
- びんちょうマグロ 70g
- 鮎の魚醤 3g
- ハーブソルト 少々
- ニンニクの芽 1 束

---

# 作り方

1. びんちょうマグロを食べやすい大きさにカットし、ハーブソルト・オリーブオイル・鮎の魚醤と和えてマリネしておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ニンニクの芽を食べやすい大きさにカットして炒める。
5. パスタがアルデンテになったらニンニクオイルの入ったフライパンを中火にかけ、フライパンにパスタとマリネしておいたマグロを加える。
6. マグロの表面が少し白みがかるまでパスタを混ぜながら加熱し、皿に盛り付ける。
