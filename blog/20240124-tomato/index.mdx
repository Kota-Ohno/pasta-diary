# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 18g
- 塩 少々
- 玉ねぎ 70g
- トマトソース 85g
- ツナ 35g
- 魚醤 4g

---

# 作り方

1. たまねぎをスライスする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. オリーブオイルを温め、玉ねぎを加えてしんなりするまで炒める。
4. ツナを加え、水分を飛ばす。
5. トマトソースを加え、具材と和え、温める。塩で調味し、酸味がきつい場合は砂糖を加える。
6. パスタがアルデンテになったらフライパンに魚醤を加え、中火にかけてパスタと和える。
7. 皿に盛り付ける。
