# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- エキストラバージンオリーブオイル 18g
- 鮎の魚醤 3g
- アスパラガス 35g
- セミドライトマト 8g
- アンチョビペースト 3g
- ペペロンチーノ(イタリア産唐辛子) 0.5g

---

# 作り方

1. 1.5%の濃度の塩水でパスタを茹でる。
2. スライスしたニンニクと唐辛子とオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
3. アスパラとセミドライトマトをフライパンに入れ、火を通す。
4. パスタがアルデンテになる直前になったら、アンチョビペーストをオイルの中で煮溶かす。
5. フライパンにパスタとアユの魚醤少々を加えて和える。
6. 皿に盛り付ける。
