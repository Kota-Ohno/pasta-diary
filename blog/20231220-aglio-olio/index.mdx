# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 15g
- にんにく 10g
- 塩 少々
- タコ 50g
- 唐辛子 少々
- アンチョビ 7g
- セミドライトマト 25g
- 大葉 5 枚

---

# 作り方

1. にんにくをスライス、タコを一口大、大葉をみじん切りにする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルと唐辛子を冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. タコとセミドライトマト、アンチョビを加え、アンチョビをオイルの中で煮溶かす。
5. パスタがアルデンテになったらゆで汁と大葉を加え、中火にかけてパスタと和える。
6. 皿に盛り付ける。
