# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- エキストラバージンオリーブオイル 18g
- 塩 2g
- 鮎の魚醤 2g
- 小松菜 70g
- アンチョビペースト 7g
- ペペロンチーノ 1 つ

---

# 作り方

1. にんにくは粗みじんに、小松菜は茎は細かめに、葉は大きめにカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ニンニクと唐辛子とオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. フライパンに小松菜の茎を入れ、火が通ったらアンチョビペーストを加える。
5. パスタがアルデンテになったら、フライパンに小松菜の葉とパスタと鮎の魚醤、ゆで汁を加え、さっと和える。
6. 皿に盛り付ける。
