# 材料

- スパゲティ 100g
- 塩 少々
- 黒胡椒 少々
- 唐辛子 微量
- パルミジャーノレッジャーノ 20g
- 玉ねぎ 60g
- ベーコン 35g
- にんにく 10g
- エキストラバージンオリーブオイル 10g

---

# 作り方

1. 玉ねぎ、にんにくをスライス、ベーコンを短冊状にカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルと唐辛子を冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ベーコンを炒め、焼き目が付いたら玉ねぎを加え、しんなりする程度に炒める。
5. パスタがアルデンテになったらゆで汁を加え、中火にかけてパスタと和える。
6. 皿に盛り付け、黒胡椒とチーズを振りかける。
