# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 15g
- にんにく 10g
- 塩 少々
- ベーコン 30g
- たまねぎ 60g
- バルサミコ酢 3g
- トマトホール 200g

---

# 作り方

1. にんにくとたまねぎをスライス、ベーコンを短冊状にカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ベーコンを入れ、焼き目を付けたら玉ねぎを加え、塩をしてしんなりするまで炒める。
5. バルサミコ酢を鍋肌に入れ、少し水分が飛んだらトマトホールを加えて煮詰める。酸味が多い場合は砂糖を加える。
6. パスタがアルデンテになったらフライパンに加え、中火にかけてパスタと和える。
7. 皿に盛り付ける。
