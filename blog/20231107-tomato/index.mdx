# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- 塩 少々
- エキストラバージンオリーブオイル 15g
- 鮎の魚醤 3g
- やりいか 100g
- プチトマト 7 個
- アンチョビペースト 5g
- 白ワイン 8g
- オレガノ 適量

# 作り方

1. にんにくをみじん切り、プチトマトは半分にカット、やりいかは掃除して目と嘴を取り、一口大にカットしておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. プチトマトを断面を下にしてフライパンに入れ、塩をして炒める。
5. プチトマトがしなっとしたら、やりいか、アンチョビペースト、白ワイン、鮎の魚醤を加え、ふたをしてイカに火を通す。
6. パスタがアルデンテになったらフライパンにオレガノを加え、中火にかけてパスタを加え和える。
7. 皿に盛り付ける。
