# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- 塩 少々
- エキストラバージンオリーブオイル 18g
- 玉ねぎ 150g
- アンチョビ 15g

---

# 作り方

1. ニンニクとたまねぎをスライスする。
2. ニンニクとたまねぎをオリーブオイルでよく炒める。
3. 1.5%の濃度の塩水でパスタを茹でる。
4. フライパンにアンチョビを加え、油に煮溶かしてよく絡める。
5. パスタがアルデンテになったらフライパンにゆで汁を加え、中火にかけてパスタと和える。
6. 皿に盛り付ける。
