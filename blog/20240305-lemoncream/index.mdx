# 材料

- スパゲティ 100g
- バター 20g
- 生クリーム 35g
- 塩 少々
- パルミジャーノレッジャーノ 15g
- ベビーホタテ 70g
- レモン 25g

---

# 作り方

1. レモンの皮を千切りに、実を絞りやすいようにカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. バターと生クリームと 20g の水を加え、バターを溶かす。
4. ベビーホタテを加えて温める。
5. パスタがアルデンテになったらレモン汁をフライパンに加え、チーズと合わせてパスタと和える。
6. 皿に盛り付け、レモンの千切りを飾る。
