# 材料

- スパゲティ 100g
- 塩 少々
- 黒胡椒 少々
- 卵 1 個
- パルミジャーノレッジャーノ 20g
- バター 20g

---

# 作り方

1. 1.5%の濃度の塩水でパスタを茹でる。
2. 目玉焼きを弱めの火で蓋をして軟らかめに焼く。
3. フライパンでバターを溶かし、パスタがアルデンテになったらゆで汁を加え、中火にかけてパスタと和える。
4. 皿に盛り付け、黒胡椒とチーズを振りかける。
