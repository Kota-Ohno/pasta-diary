# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 18g
- にんにく 10g
- 塩 少々
- 唐辛子 少々
- 魚醤 4g
- アサリむき身 60g
- 白ワイン 15g
- ミニトマト 60g

---

# 作り方

1. にんにくをスライス、ミニトマトを半分にカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルと唐辛子を冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ミニトマトを焼く。
5. アサリと白ワインを加え、アルコールを飛ばす。
6. パスタがアルデンテになったらフライパンに魚醤を加え、中火にかけてパスタと和える。
7. 皿に盛り付ける。
