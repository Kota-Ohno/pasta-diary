# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- 塩 少々
- エキストラバージンオリーブオイル 20g
- 鮎の魚醤 4g
- ひらたけ 80g
- どんこ 70g
- ペペロンチーノ 半かけ
- イタリアンパセリ 適量

# 作り方

1. にんにくをスライス、ひらたけとどんこは食べやすい大きさにカットしておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ひらたけとどんこを加え、塩をして炒める。炒まったらイタリアンパセリとペペロンチーノを加える。
5. パスタがアルデンテになったらフライパンに鮎の魚醤を加え、中火にかけてパスタを加えて和える。
6. 皿に盛り付ける。
