# 材料

- スパゲティ 100g
- ニンニク 10g
- バター 15g
- 生クリーム 80g
- 塩 少々
- パルミジャーノレッジャーノ 15g
- ほうれん草 50g
- エビ 75g
- イカ 50g
- 白胡椒 0.5g

---

# 作り方

1. イカとほうれん草を一口大にカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. エビとイカを炒める。
4. バターと生クリームを加え、バターを溶かす。白コショウと塩を加えて調味する。
5. パスタが上がる 1 分前にほうれん草を鍋に加えて茹でる。
6. パスタがアルデンテになったらほうれん草とパスタをフライパンに加え、チーズと合わせてパスタと和える。
7. 皿に盛り付ける。
