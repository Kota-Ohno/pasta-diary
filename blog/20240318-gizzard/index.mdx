# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 10g
- にんにく 10g
- 塩 少々
- 胡椒 少々
- ニンニクの芽 30g
- 砂肝 90g

---

# 作り方

1. にんにくをスライス、砂肝を一口大、ニンニクの芽をそぎ切りでカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. 砂肝とニンニクの芽を加え、塩と胡椒をして炒める。
5. パスタがアルデンテになったらフライパンにゆで汁を加え、中火にかけてパスタと和える。
6. 皿に盛り付ける。
