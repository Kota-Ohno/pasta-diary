# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 10g
- トマトソース 100g
- モッツァレラチーズ 50g
- スイートバジル 10g

---

# 作り方

1. 1.5%の濃度の塩水でパスタを茹でる。
2. モッツァレラチーズをサイコロ状にカットする。
3. フライパンでトマトソースを加熱する。
4. パスタがアルデンテになったら、フライパンにパスタを加え、飾り用以外のモッツァレラチーズとちぎったバジルを入れる。
5. 軽くチーズが解けるまで和えて、皿に盛り付け、飾り用のモッツァレラチーズをのせる。
