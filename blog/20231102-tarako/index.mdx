# 材料

- スパゲティ 100g
- たらこ 50g
- めんつゆ(創味のつゆ) 6g
- エキストラバージンオリーブオイル 8g
- バター 15g
- 大葉 適量
- 刻みのり 適量
- 塩昆布 適量

---

# 作り方

1. たらこは皮が気にならなくなる程度に叩いて、大葉は刻んでおく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ボウルにたらこ、オリーブオイル、めんつゆ、バターを加える。
4. パスタ湯の蒸気でボウルを温め、バターを溶かす。
5. パスタがアルデンテになったらボウルにパスタを加えて和える。
6. 皿に盛り付け、刻みのり、塩昆布、大葉を飾る。
