# 材料

- スパゲティ 100g
- ニンニク 10g
- エキストラバージンオリーブオイル 5g
- 生クリーム 30g
- 有塩バター 20g
- 塩 1g
- たまねぎ 50g
- 舞茸 50g
- トマトソース 50g
- 黒胡椒 0.5g

# 作り方

1. たまねぎとにんにくをスライスする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. たまねぎを入れ、しんなりするくらいまで炒めた後、舞茸も投入して炒める。
5. バターと生クリームとトマトソースを入れ、バターを溶かしておく。(煮詰めない)
6. パスタがアルデンテになったらフライパンにパスタを入れ、黒胡椒を加えて和える。
7. 皿に盛り付ける。
