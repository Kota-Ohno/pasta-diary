# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- 塩 少々
- エキストラバージンオリーブオイル 18g
- だし醤油 7g
- ひらたけ 40g
- いわし 130g
- 長ネギ 15g
- ゆず 半分

---

# 作り方

1. にんにくをスライス、ゆずは半分にして種を取り、イワシは三枚おろしにして皮をむいて塩をしておき、ひらたけは食べやすい大きさにカットしておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. イワシを焼き、1 枚飾り用によけておく。
5. ひらたけを加え、しんなりしたらネギを加える。
6. パスタがアルデンテになったらフライパンにだし醤油を加え、中火にかけてパスタとゆで汁を加えて和える。
7. 皿に盛り付け、飾り用のいわしとゆずを添える。
