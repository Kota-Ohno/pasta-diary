# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 15g
- にんにく 20g
- 塩 少々
- 新たまねぎ 100g
- プロシュート 2 枚
- ルッコラ 20g

---

# 作り方

1. にんにくと新たまねぎをスライスする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. 新たまねぎを入れ、食感が残る程度に炒める。
5. パスタがアルデンテになったらフライパンにゆで汁を加え、中火にかけてパスタと和える。
6. 皿に盛り付け、ルッコラと生ハムをのせる。
