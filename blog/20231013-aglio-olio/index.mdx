# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- エキストラバージンオリーブオイル 18g
- 塩 2g
- 鮎の魚醤 3g
- しいたけ 70g
- パルミジャーノレッジャーノ 15g

---

# 作り方

1. にんにくは粗みじんに、しいたけは傘をスライス、軸は細かいサイコロ状にカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ニンニクとしいたけの軸、オリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. フライパンにしいたけの傘を入れ、さっと火を通す。
5. パスタがアルデンテになったら、フライパンにパスタと鮎の魚醤、ゆで汁を加え、さっと和える。
6. 皿に盛り付け、パルミジャーノレッジャーノを振りかける。
