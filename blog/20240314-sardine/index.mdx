# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 15g
- にんにく 10g
- 塩 少々
- 黒胡椒 少々
- オイルサーディン 35g
- 白だし 5g
- レモン汁 5g
- 醤油 5g
- 長ネギ 30g

---

# 作り方

1. にんにくとネギをスライスする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. オイルサーディンを加え、焼き色を付ける。
5. ネギを加えてしんなりしたら、黒胡椒、白だし、レモン汁、しょうゆを加え、火を止めておく。
6. パスタがアルデンテになったらフライパンにゆで汁を加え、中火にかけてパスタと和える。
7. 皿に盛り付ける。
