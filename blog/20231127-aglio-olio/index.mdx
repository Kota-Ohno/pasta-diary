# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- 塩 少々
- エキストラバージンオリーブオイル 18g
- なめこ 30g
- まいたけ 40g
- ブナピー 40g
- 白だし 7g
- ゆず七味 適量

# 作り方

1. ニンニクをみじん切りに、キノコ類は食べやすい大きさにちぎっておく。
2. まいたけとブナピーを油を敷いていないフライパンで乾煎りして水分を飛ばす。
3. 1.5%の濃度の塩水でパスタを茹でる。
4. ニンニクとオリーブオイルを冷えた状態の別のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
5. なめこを入れ、ぬめりが焦げ付く程度までしっかりと炒める。
6. 乾煎りしておいたまいたけとブナピー、白だしを加え、火を止めておく。
7. パスタがアルデンテになったらフライパンにゆで汁を加え、中火にかけてパスタと和える。
8. 皿に盛り付ける。お好みでゆず七味を振りかける。
