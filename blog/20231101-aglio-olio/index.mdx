# 材料

- スパゲティ 100g
- ニンニク 1 かけ
- 塩 少々
- エキストラバージンオリーブオイル 15g
- 鮎の魚醤 3g
- アスパラ 35g
- 生ハム 3 枚
- 大葉 適量

---

# 作り方

1. にんにくをスライス、アスパラは一口大にカット、大葉は刻んでおく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. アスパラを炒め、塩を軽く振っておく。
5. パスタがアルデンテになったらフライパンに鮎の魚醤を加え、中火にかけてパスタを加えて和える。
6. 皿に盛り付け、生ハムと大葉をトッピングする。
