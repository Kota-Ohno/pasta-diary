# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 18g
- 塩 少々
- ベーコン 45g
- トマト 150g
- ツナ 20g
- アンチョビ 7g
- ニンニク 10g
- オレガノ 少々

---

# 作り方

1. たまねぎ、ニンニクをスライス、ベーコンを短冊切りにする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ベーコンを入れ、焼き目をつける。
5. ツナを加え、水分を飛ばす。
6. トマトを加え、崩れる程度まで炒める。塩とアンチョビで調味し、酸味がきつい場合は砂糖を加える。
7. パスタがアルデンテになったらフライパンに加え、中火にかけてパスタと和える。
8. 皿に盛り付け、オレガノを振りかける。
