# 材料

- 2.2mm 太麺スパゲティ 100g
- にんにく 1 かけ
- 塩 少々
- エキストラバージンオリーブオイル 10g
- たまねぎ 30g
- ケチャップ 50g
- ウスターソース 7g
- はちみつ 4g
- ソーセージ 30g
- ベーコン 35g
- ピーマン 30g

# 作り方

1. にんにくをみじん切り、たまねぎとピーマンをスライス、ベーコンを短冊に、ソーセージを斜めに切っておく。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. ベーコンとソーセージを加え、焦げ目が適度につくように焼く。
5. たまねぎとピーマンを加え、塩をしてしんなりするくらいまで炒める。
6. 鍋肌にケチャップ、ウスターソース、はちみつを入れ、しばらく焼いて水分を飛ばしたら具材と和える。
7. パスタがアルデンテになったらフライパンを中火にかけて、ゆで汁とパスタを加え和える。
8. 皿に盛り付ける。お好みでタバスコや卵焼きを添えてもおいしい。
