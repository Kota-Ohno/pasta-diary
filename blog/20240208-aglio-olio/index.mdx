# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 15g
- にんにく 10g
- 塩 少々
- プロシュート 2 枚

---

# 作り方

1. にんにくをスライスする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. パスタがアルデンテになったらフライパンに加え、中火にかけてパスタと和える。
5. 皿に盛り付け、生ハムをのせる。
