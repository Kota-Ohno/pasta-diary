# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 8g
- 卵 1 個
- パルミジャーノレッジャーノ 25g
- 黒胡椒 1g
- たまねぎ 30g
- ベーコン 50g

# 作り方

1. ベーコンは短冊状に、玉ねぎはみじん切りにカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. ベーコンをオリーブオイルで炒め、ベーコンの油が出てきたら玉ねぎを加えてさらに炒める。
4. ボウルに卵とすりおろしたパルミジャーノレッジャーノを入れ、よく混ぜておく。
5. パスタがアルデンテになったら、フライパンにパスタを加えて火を止める。
6. 卵液をパスタの上にかけて、ごく弱火でチーズが溶けてソースがちょうどよい粘度になるまで混ぜながら加熱する。
7. 皿に盛り付けて砕いた黒胡椒を振りかける。
