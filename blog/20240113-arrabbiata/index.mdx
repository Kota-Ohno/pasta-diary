# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 18g
- にんにく 15g
- 塩 少々
- 菜の花 80g
- 唐辛子 1.5 かけ
- トマトソース 50g

---

# 作り方

1. にんにくをスライス、菜の花をある程度の長さにカットする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルと唐辛子を冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. トマトソースを加え、軽く煮詰める。
5. パスタが上がる 1 分前に菜の花を鍋に加えてボイルする。
6. パスタがアルデンテになったらフライパンにゆで汁を加え、中火にかけてパスタと和える。
7. 皿に盛り付ける。
