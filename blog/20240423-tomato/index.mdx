# 材料

- スパゲティ 100g
- エキストラバージンオリーブオイル 15g
- 塩 少々
- ニンニク 10g
- 魚醤 4g
- ビンチョウマグロ 90g
- たまねぎ 30g
- トマトホール 200g

---

# 作り方

1. ニンニク、たまねぎをみじん切り、ビンチョウマグロを角切りにする。
2. 1.5%の濃度の塩水でパスタを茹でる。
3. スライスしたニンニクとオリーブオイルを冷えた状態のフライパンに入れ、加熱しニンニクの香りをオイルに移す。
4. トマトホールを加えて、固形部分を崩す。塩と魚醤で調味し、酸味がきつい場合は砂糖を加える。
5. パスタがアルデンテになったらフライパンに加え、中火にかけてパスタと和える。
6. 皿に盛り付ける。
